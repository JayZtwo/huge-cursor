Install-and-use macOS utility for Codex users

Shake your mouse. Get a Codex fortune that never repeats.

Install Shake Cursor, hide it in the background, then shake anywhere. A lightweight Codex assistant appears near your cursor to draw a realtime fortune, answer a daily question, or create a calendar event from plain text.

Download DMG View source

No custom backend. Fortunes are generated by your local Codex session from time, cursor trigger, and available memory.

Apple should probably build this into Spotlight.

Install and use

Open once, hide to background, then shake in any direction to summon it.

No preset fortunes

Each fortune is generated in realtime from the current moment and available memory.

Daily assistant

Ask, reflect, or create macOS Calendar events with clear time text.
