## Short Description

Shake Cursor is an open-source macOS utility for Codex users. Install it, hide it in the background, then shake your mouse anywhere to summon a lightweight local Codex assistant near the cursor. It generates each fortune in realtime from the current moment, cursor trigger, and available memory instead of using preset copy, answers daily questions, and writes clearly timed plans into macOS Calendar.

No custom backend. No developer server. The app uses your local Codex session and local macOS permissions.

First comment:

> I built Shake Cursor because the cursor is the fastest place to start thinking.
>
> macOS already has a shake gesture for finding the pointer. I wanted to turn that instinct into an AI entry point: install once, hide it in the background, then shake, type, and get help from local Codex.
>
> The fortune part is not a preset text library. Every draw asks Codex to generate a fresh short fortune from the current time, cursor trigger, and available life memory, so it should feel tied to this moment instead of mechanically repeated.
>
> V1 supports:
> - full-direction mouse shake detection
> - local Codex Desktop bridge
> - short fortune-style responses
> - daily assistant prompts
> - macOS Calendar creation when the input contains clear time
>
> It is open source, notarized, and has no custom backend.
>
> The slightly provocative pitch: Apple should probably build this into Spotlight.
