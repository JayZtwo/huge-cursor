## Short Description

Shake Cursor is an open-source macOS utility for Codex users. Shake your mouse anywhere, and a lightweight input appears near the cursor. It talks to your local Codex Desktop session, can generate a short ritual-style fortune, answer daily questions, and write clearly timed plans into macOS Calendar.

No custom backend. No developer server. The app uses your local Codex session and local macOS permissions.
